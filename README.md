Flag Capture Game (Pygame)

Project Overview:
Flag Capture Game is a simple 2D game built using Python and Pygame.
The player controls a red circle and must capture all the flags placed randomly inside the play area. Each captured flag increases the score, and the game ends once all flags are collected.
This project was created to understand game loops, collision detection, keyboard handling, and rendering graphics using Pygame.

Gameplay:
Move the red circle using arrow keys
Capture all 6 colored flags
Each flag captured gives 50 points
The game ends when all flags are collected

Tech Stack:
Language: Python

Library: Pygame
